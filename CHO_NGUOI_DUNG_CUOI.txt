📦 CÀI ĐẶT

1. Download file "VietnameseOCRTool_Setup.exe"
2. Double-click để chạy
3. Nếu Windows hiện cảnh báo → Xem phần "Windows Defender" bên dưới
4. Làm theo hướng dẫn cài đặt
5. Sau khi cài xong, app sẽ chạy tự động


🚀 SỬ DỤNG

1. Sau khi cài đặt, tìm icon "OCR" trên System Tray (góc dưới bên phải)
   
2. Lần đầu chạy:
   - App sẽ hỏi bạn muốn dùng phím tắt nào
   - Nhấn tổ hợp phím bạn thích (ví dụ: Ctrl+Shift+O)
   - Phím tắt sẽ được lưu lại

3. Khi cần OCR:
   - Nhấn phím tắt đã đặt
   - Kéo chuột chọn vùng có chữ
   - Nhả chuột → Văn bản tự động copy vào clipboard
   - Paste (Ctrl+V) ở bất kỳ đâu!

4. Thay đổi phím tắt:
   - Chuột phải vào icon trên System Tray
   - Chọn "Thay đổi phím tắt"
   - Nhấn tổ hợp phím mới

5. Thoát ứng dụng:
   - Chuột phải vào icon trên System Tray
   - Chọn "Thoát"

CÁCH KHẮC PHỤC (Chọn 1 trong 3):

┌─────────────────────────────────────────────────────────┐
│ CÁCH 1: Click "More info" rồi "Run anyway"             │
└─────────────────────────────────────────────────────────┘

Khi Windows hiện cảnh báo:
1. Click vào "More info" hoặc "Thông tin thêm"
2. Click "Run anyway" hoặc "Vẫn chạy"
3. App sẽ cài đặt bình thường


┌─────────────────────────────────────────────────────────┐
│ CÁCH 2: Thêm Exclusion (Khuyến nghị)                   │
└─────────────────────────────────────────────────────────┘

1. Mở Windows Security (Windows Defender)
   - Start → gõ "Windows Security"
   
2. Chọn "Virus & threat protection"

3. Scroll xuống → "Manage settings"

4. Scroll xuống → "Exclusions"
   - Click "Add or remove exclusions"
   
5. Click "Add an exclusion" → "File" hoặc "Folder"

6. Chọn file "VietnameseOCRTool_Setup.exe"
   hoặc folder sau khi cài: "C:\Program Files\VietnameseOCRTool\"

7. Cài đặt lại file setup


┌─────────────────────────────────────────────────────────┐
│ CÁCH 3: Tạm tắt Real-time Protection                   │
└─────────────────────────────────────────────────────────┘

1. Mở Windows Security

2. Virus & threat protection

3. Manage settings

4. Tắt "Real-time protection" (tạm thời)

5. Cài đặt app

6. Bật lại "Real-time protection"


⚠️ LƯU Ý QUAN TRỌNG:

→ File này KHÔNG PHẢI virus
→ Bạn hoàn toàn an toàn khi sử dụng
→ Nếu không tin, có thể kiểm tra tại: https://www.virustotal.com/
→ Đây là vấn đề nổi tiếng của Windows với các app Python


💡 TIPS SỬ DỤNG

✓ Chọn vùng có chữ rõ ràng, nền đơn giản
✓ Tránh chọn vùng có nhiều hình ảnh phức tạp

🐛 GẶP VẤN ĐỀ?

Nếu app không hoạt động:

1. Khởi động lại app
   - Thoát hoàn toàn (chuột phải icon → Thoát)
   - Chạy lại từ Start Menu

2. Kiểm tra phím tắt có bị trùng không
   - Thử đổi sang tổ hợp phím khác

3. Chạy lại file cài đặt
   - Chọn "Repair" hoặc "Uninstall" rồi cài lại

4. Kiểm tra Windows Event Viewer để xem log
